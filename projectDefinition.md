# Tour Planner App

### An app that lets a user plan how long an outing will take depending on what type of terrain they are in.

#### Task Analysis
* Create Account
* Login
* Logout  

* Munter Calculation  

* Create a Tour Plan
* Read Tour Plans
* Update a Tour Plan
* Delete a Tour Plan  

#### User Persona
* Outdoorsy person of any age that wants to know how long a tour will take.
